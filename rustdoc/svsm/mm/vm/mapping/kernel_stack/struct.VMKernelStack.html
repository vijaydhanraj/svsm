pub struct VMKernelStack {
    alloc: RawAllocMapping,
    guard_pages: usize,
}
Expand description

Mapping to be used as a kernel stack. This maps a stack including guard pages at the top and bottom.

Fields§

§alloc: RawAllocMapping

Allocation for stack pages
